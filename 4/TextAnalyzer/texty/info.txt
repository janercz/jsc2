This folder contains realistic sample files for parser and analyzer tests.
The content is intentionally mixed: plain text, XML, CSV, JSON, markdown, and code snippets.

Primary goals:
- Provide human readable data for tokenization and indexing checks.
- Include short and long lines to test splitting logic.
- Keep examples deterministic for repeatable test runs.
